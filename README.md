# Glassmorphism UI Design

A modern user interface project showcasing the Glassmorphism design trend. This project focuses on visual aesthetics, utilizing advanced CSS blur effects and transparency to create a frosted-glass look.

---

## Live Deployment
[Visit the Live Design](https://kanidehbokri.github.io/glassmorphism-login-ui/)

---

## Technical Features
* Glassmorphism Effects: Implementation of backdrop-filter and semi-transparent backgrounds for modern UI depth.
* Modern Typography: Carefully selected font weights and spacing for high readability.
* Responsive Layout: Optimized for all screen sizes to maintain visual integrity.
* CSS Architecture: Use of variables and clean classes for manageable styling.

---

## Core Technologies
* HTML5 - Semantic structure.
* CSS3 - Backdrop filters, linear gradients, and responsive design.
